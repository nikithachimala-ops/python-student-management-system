STUDENT MANAGEMENT SYSTEM
A Simple and efficient "Student Mnagement System" built using "Python" for managing student records through a command-line interface.
this project allows users to add,view,update,and delete student information while storing data using CSV files.

FEATURES-
Adding new student records
view all settings
update existing student details
Delete student records
store data using CSV files
simple and user-friendly interface

TECHNOLOGIES USED-
Python 3
CSV File Handling
Object Oriented Programming(OOP)

Project Structure-
Python-student-management-system/
   main.py
   student_management.py
   students.csv
   README.md
   LICENSE

HOW TO RUN:
#Clone the Repository

'''bash
git clone https://github.com/nikithachimala-ops/python-student-management-system.git
'''

#Navigate to Project Directory

'''bash
cd python-student-management-system
'''
#Run the Application
'''bash
python main.py
'''
